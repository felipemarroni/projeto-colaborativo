# Projeto-colaborativo
Repositório de projetos para treinamento da FGV. Foi utilizado o VS Code como a IDE para esta tarefa, devido a existência de ferramentas que integrem com o GitHub e facilitam a realização das tarefas.

# 3. Commit

<img width="1854" height="1048" alt="commit" src="https://github.com/user-attachments/assets/e6ccc845-6ca3-4b57-986d-dc27709c05bd" />

Criação do commit, usando o parâmetro "-a" para commitar todos os arquivos que já estavam tracked. Para isso, antes de executar esse comando, foi usado o comando "git add .", para fazer o staging de todos os arquivos não-trackeados.

# 4. Push da branch local

<img width="1854" height="1048" alt="push" src="https://github.com/user-attachments/assets/422d2290-f696-4b70-82e5-a3e7bcfb4598" />

Comando para fazer o push da branch local para o remoto (criando a branch no repositório remoto). O parãmetro "-u" foi usado para que a branch remota seja sempre o alvo quando o comando push for dado na branch local, simplificando futuros pushs.
